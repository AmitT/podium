### Install gulp

Building the theme requires [node.js](http://nodejs.org/download/). We recommend you update to the latest version of NPM: [`npm i -g npm`](https://docs.npmjs.com/try-the-latest-stable-version-of-npm).

From the command line:

1. Navigate to the theme directory, then run `npm install`
2. Change your site URL in webpack.config.js file proxy.

### Build commands

* `npm start` — Compile assets & start BrowserSync session (listening to file changes, recompile & push changes to your browser).
* `npm run build:dev` — Compile and optimize the files in your assets directory
* `npm run build` OR `npm run build:production` — Compile minified assets for production

You now have all the necessary dependencies to run the build process.
